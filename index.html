Parte I

FEM & Problema de Poisson

El escenario modelo del análisis numérico: un dominio, una EDP elíptica, una condición de contorno y una maquinaria variacional elegante.

Problema modelo

Dado $\Omega \subset \mathbb{R}^n$ abierto acotado con frontera $\partial\Omega$ Lipschitz, y $f \in L^2(\Omega)$, buscamos $u: \Omega \to \mathbb{R}$ tal que:

Problema de Poisson — Forma Fuerte $$-\Delta u = f \quad \text{en } \Omega$$ $$u = 0 \quad \text{en } \partial\Omega$$

donde $\Delta u = \nabla \cdot (\nabla u) = \sum_{i=1}^n \partial_{x_i}^2 u$ es el operador de Laplace. La condición $u=0$ en $\partial\Omega$ se llama condición de Dirichlet homogénea.

¿Qué es el MEF?
1

Triangulación del dominio

Se divide $\Omega$ en elementos simples (triángulos en 2D, tetraedros en 3D) formando la malla $\mathcal{T}_h$, con parámetro de malla $h = \max_K \text{diam}(K)$.

2

Funciones de base locales

Polinomios $\varphi_i$ de soporte compacto ($\text{supp}(\varphi_i) \subset$ estrella del nodo $i$). Para P1: $\varphi_i(x_j) = \delta_{ij}$.

3

Ensamblaje y resolución

Se construye el sistema global $\mathbf{K}\mathbf{U} = \mathbf{F}$ y se resuelve con factorización de Cholesky o métodos iterativos (CG, GMRES).

De la forma fuerte a la forma débil

Multiplicamos la EDP por una función test $v \in V = H_0^1(\Omega)$ e integramos sobre $\Omega$:

Paso 1 — Multiplicar e integrar $$\int_\Omega (-\Delta u)\, v\, dx = \int_\Omega f\, v\, dx$$
Paso 2 — Integración por partes (identidad de Green) $$\int_\Omega \nabla u \cdot \nabla v\, dx \underbrace{- \int_{\partial\Omega} \frac{\partial u}{\partial n}\, v\, d\sigma}_{=0 \text{ pues } v|_{\partial\Omega}=0} = \int_\Omega f\, v\, dx$$
Formulación débil — Forma compacta $$\text{Hallar } u \in V = H_0^1(\Omega) \text{ tal que:}$$ $$a(u,v) = \ell(v) \qquad \forall\, v \in V$$ $$\text{donde} \quad a(u,v) = \int_\Omega \nabla u \cdot \nabla v\, dx, \quad \ell(v) = \int_\Omega f\, v\, dx$$

El espacio de Sobolev $H_0^1(\Omega)$ se define como: $$H_0^1(\Omega) = \left\{ v \in L^2(\Omega)\;:\; \nabla v \in [L^2(\Omega)]^n,\; v|_{\partial\Omega} = 0 \right\}$$ dotado del producto interno $\langle u,v\rangle_{H^1} = \int_\Omega \nabla u \cdot \nabla v\, dx$ (equivalente por Poincaré).

Existencia y unicidad continua

Continuidad de $a$: Existe $M > 0$ tal que $$|a(u,v)| \leq M\, \|u\|_{H^1}\, \|v\|_{H^1} \quad \forall\, u,v \in V$$ (directa por Cauchy-Schwarz).

Coercividad de $a$: Por la desigualdad de Poincaré $\|v\|_{L^2} \leq C_\Omega \|\nabla v\|_{L^2}$, existe $\alpha > 0$ tal que $$a(v,v) = \|\nabla v\|_{L^2}^2 \geq \alpha\, \|v\|_{H^1}^2 \quad \forall\, v \in V$$

Lax-Milgram: Sea $V$ un espacio de Hilbert, $a:V\times V\to\mathbb{R}$ bilineal continua y coerciva, y $\ell \in V'$ lineal continua. Entonces existe un único $u \in V$ tal que $$a(u,v) = \ell(v) \quad \forall\, v \in V$$ y además $\|u\|_V \leq \frac{1}{\alpha}\|\ell\|_{V'}$.

Para el problema de Poisson: $V = H_0^1(\Omega)$ es Hilbert ✓, $a$ continua ($M=1$) ✓, $a$ coerciva ✓, $\ell(v) = (f,v)_{L^2}$ continua ✓. Por tanto $\exists!\, u \in H_0^1(\Omega)$. $\square$

Esquema de Galerkin — Nivel discreto

Espacio discreto P1: $$V_h = \left\{ v_h \in C^0(\bar\Omega)\;:\; v_h|_K \in \mathbb{P}_1(K)\;\forall K \in \mathcal{T}_h,\; v_h|_{\partial\Omega}=0 \right\}$$ con $\dim(V_h) = N_h$ (número de nodos interiores). Se tiene $V_h \subset H_0^1(\Omega)$.

Problema discreto (Galerkin) $$\text{Hallar } u_h \in V_h \text{ tal que:}$$ $$a(u_h, v_h) = \ell(v_h) \quad \forall\, v_h \in V_h$$
Representación en base $\{\varphi_i\}$ $$u_h = \sum_{j=1}^{N_h} U_j\, \varphi_j$$ $$\sum_{j=1}^{N_h} U_j\, a(\varphi_j, \varphi_i) = \ell(\varphi_i) \quad \forall\, i$$ $$\Longrightarrow \quad \mathbf{K}\,\mathbf{U} = \mathbf{F}$$

Existencia y unicidad discreta (inmediata): $V_h \subset H_0^1(\Omega)$ es subespacio cerrado de Hilbert, hereda continuidad y coercividad de $a$. Lax-Milgram aplica en $V_h$ directamente. $\mathbf{K}$ es SPD. $\square$

Sistema lineal, ensamblaje y convergencia
Entradas del sistema $$K_{ij} = \int_\Omega \nabla\varphi_i \cdot \nabla\varphi_j\, dx \quad \text{(rigidez)}$$ $$F_i = \int_\Omega f\,\varphi_i\, dx \quad \text{(carga)}$$ $$\mathbf{K} \text{ es dispersa, simétrica, definida positiva}$$

La matriz $\mathbf{K}$ tiene $\mathcal{O}(N_h)$ entradas no nulas (soporte local de $\varphi_i$). Se ensambla elemento por elemento: $$K_{ij} = \sum_{K \in \mathcal{T}_h} \int_K \nabla\varphi_i \cdot \nabla\varphi_j\, dx$$

Estimación de Céa: $$\|u - u_h\|_{H^1} \leq \frac{M}{\alpha}\, \inf_{v_h \in V_h} \|u - v_h\|_{H^1}$$ $u_h$ es la mejor aproximación en $V_h$, salvo constante $M/\alpha$.

Convergencia P1 (Aubin-Nitsche): $$\|u - u_h\|_{H^1(\Omega)} \leq C\, h\, |u|_{H^2(\Omega)}$$ $$\|u - u_h\|_{L^2(\Omega)} \leq C\, h^2\, |u|_{H^2(\Omega)}$$

Parte III

Extensiones del Problema Modelo

Más allá de Dirichlet homogéneo: condiciones de Neumann, mixtas, el operador de difusión general y el problema de Stokes.

Condición de Neumann
Formulación fuerte $$-\Delta u = f \quad \text{en } \Omega$$ $$\frac{\partial u}{\partial n} = g \quad \text{en } \partial\Omega$$

donde $\partial u/\partial n = \nabla u \cdot \mathbf{n}$ es la derivada normal exterior. Modela flujo prescrito en la frontera (flujo de calor, corriente eléctrica).

Condición de compatibilidad: Para existencia es necesario: $$\int_\Omega f\, dx + \int_{\partial\Omega} g\, d\sigma = 0$$ (balance global de flujo).

Formulación débil — Neumann natural $$\text{Hallar } u \in H^1(\Omega)/\mathbb{R} \text{ t.q.:}$$ $$\int_\Omega \nabla u \cdot \nabla v\, dx = \int_\Omega f\, v\, dx + \int_{\partial\Omega} g\, v\, d\sigma \quad \forall v$$

La condición de Neumann es natural: surge automáticamente de la integración por partes, sin imponerse sobre el espacio de funciones test. El término de borde $\int_{\partial\Omega} g\, v\, d\sigma$ aparece directamente en $\ell(v)$.

Condición mixta — Robin
Formulación fuerte $$-\Delta u = f \quad \text{en } \Omega, \qquad \alpha u + \beta\frac{\partial u}{\partial n} = g \quad \text{en } \partial\Omega \quad (\alpha,\beta \geq 0,\; \alpha+\beta>0)$$
Formulación débil $$a(u,v) = \int_\Omega \nabla u \cdot \nabla v\, dx + \frac{\alpha}{\beta}\int_{\partial\Omega} u\, v\, d\sigma$$ $$\ell(v) = \int_\Omega f\,v\, dx + \frac{1}{\beta}\int_{\partial\Omega} g\, v\, d\sigma$$

Con $\alpha > 0$: el término de borde $\int_{\partial\Omega} u\,v$ aporta coercividad adicional. Lax-Milgram aplica en $V = H^1(\Omega)$ (sin restricción en la frontera). Ejemplos: convección-difusión, Newton cooling.

Difusión variable — operador general
EDP de difusión general $$-\nabla \cdot \bigl(\mathbf{A}(x)\,\nabla u\bigr) + \mathbf{b}(x)\cdot\nabla u + c(x)\,u = f$$

con $\mathbf{A}(x) \in \mathbb{R}^{n\times n}$ tensor de difusión (simétrico, uniformemente elíptico), $\mathbf{b}$ campo de convección, $c \geq 0$ reacción.

Elipticidad uniforme: $\exists\, \lambda > 0$ tal que $\boldsymbol{\xi}^T \mathbf{A}(x)\,\boldsymbol{\xi} \geq \lambda\,|\boldsymbol{\xi}|^2$ para casi todo $x \in \Omega$ y todo $\boldsymbol{\xi} \in \mathbb{R}^n$.

Formulación débil correspondiente $$a(u,v) = \int_\Omega \mathbf{A}\nabla u \cdot \nabla v\, dx + \int_\Omega (\mathbf{b}\cdot\nabla u)\,v\, dx + \int_\Omega c\,u\,v\, dx$$

Bajo las condiciones de elipticidad y $\mathbf{b}\in [L^\infty]^n$, $c \in L^\infty$, $c - \tfrac{1}{2}\nabla\cdot\mathbf{b} \geq 0$: la forma bilineal es continua y coerciva → Lax-Milgram aplica.

Problema de Stokes — fluidos viscosos incompresibles
Sistema de Stokes $$-\mu\,\Delta\mathbf{u} + \nabla p = \mathbf{f} \quad \text{en } \Omega$$ $$\nabla \cdot \mathbf{u} = 0 \quad \text{en } \Omega$$ $$\mathbf{u} = \mathbf{0} \quad \text{en } \partial\Omega$$

$\mathbf{u}$ es la velocidad del fluido, $p$ la presión, $\mu$ la viscosidad dinámica. La incompresibilidad $\nabla\cdot\mathbf{u}=0$ introduce una restricción.

Formulación débil — Espacio mixto $$\text{Hallar } (\mathbf{u},p) \in [H_0^1]^n \times L_0^2 \text{ t.q.:}$$ $$\mu\int_\Omega \nabla\mathbf{u}:\nabla\mathbf{v}\, dx - \int_\Omega p\,\nabla\cdot\mathbf{v}\, dx = \int_\Omega \mathbf{f}\cdot\mathbf{v}\, dx$$ $$\int_\Omega q\,\nabla\cdot\mathbf{u}\, dx = 0$$

Condición inf-sup (Babuška-Brezzi): $$\inf_{q \in L_0^2} \sup_{\mathbf{v}\in [H_0^1]^n} \frac{\int_\Omega q\,\nabla\cdot\mathbf{v}}{\|q\|_{L^2}\|\nabla\mathbf{v}\|_{L^2}} \geq \beta > 0$$ garantiza existencia y unicidad del par $(\mathbf{u},p)$. Los elementos deben satisfacer la condición inf-sup discreta (Taylor-Hood: P2/P1).

El proceso completo

Del modelo matemático a FreeFEM++ — el viaje completo del análisis numérico, incluida la parte cómica.

fem-poisson.edp — FEM Pipeline
📝
1. El Modelo Matemático
Tenemos un fenómeno físico — temperatura, potencial eléctrico, deflexión… Lo traducimos en una ecuación diferencial parcial bien definida.
$$-\Delta u = f \quad \text{en } \Omega \subset \mathbb{R}^2, \qquad u = 0 \quad \text{en } \partial\Omega$$
🔬
2. Formulación Variacional Débil (continua)
Multiplicamos por $v \in H_0^1(\Omega)$, integramos, y aplicamos Green. La solución clásica se relaja a una solución débil.
$$\int_\Omega \nabla u \cdot \nabla v\, dx = \int_\Omega f\, v\, dx \qquad \forall\, v \in H_0^1(\Omega)$$
✅
3. Existencia y Unicidad — Nivel Continuo
$H_0^1(\Omega)$ es un espacio de Hilbert. $a(u,v) = \int\nabla u\cdot\nabla v$ es continua y coerciva. El Lema de Lax-Milgram garantiza la solución única.
$$\underbrace{H_0^1(\Omega)}_{\text{Hilbert}} + \underbrace{a \text{ coerciva}}_{\alpha\|v\|^2 \leq a(v,v)} + \underbrace{\ell \text{ continua}}_{|\ell(v)|\leq C\|v\|} \;\Longrightarrow\; \exists!\, u \in H_0^1(\Omega)$$
🔲
4. Discretización por Galerkin
Triangulamos $\Omega$ con malla $\mathcal{T}_h$. Construimos $V_h \subset H_0^1(\Omega)$ de dimensión finita con polinomios P1 en cada triángulo.
$$V_h = \text{span}\{\varphi_1,\ldots,\varphi_{N_h}\}, \quad u_h = \sum_j U_j\,\varphi_j$$ $$\Longrightarrow \quad \mathbf{K}\,\mathbf{U} = \mathbf{F}, \quad K_{ij} = \int_\Omega \nabla\varphi_i\cdot\nabla\varphi_j\,dx$$
⚡
5. Existencia y Unicidad Discreta — ¡Inmediata!
Como $V_h \subset H_0^1(\Omega)$ hereda la estructura de Hilbert y las propiedades de $a$, Lax-Milgram aplica automáticamente en $V_h$.
$$V_h \subset H_0^1(\Omega) \Rightarrow \text{Hilbert} \checkmark \quad a|_{V_h} \text{ coerciva} \checkmark$$ $$\Longrightarrow \quad \mathbf{K} \text{ es SPD} \Longrightarrow \mathbf{K}\,\mathbf{U}=\mathbf{F} \text{ tiene } \exists!\text{ solución} \;\square$$

// ─── Problema de Poisson · MEF · FreeFEM++ ───────────────
// -Δu = f en [0,1]²,  u = 0 en ∂Ω
// Solución exacta: u = sin(πx)sin(πy)
// f = 2π²sin(πx)sin(πy)

// 1. Construir la malla triangular
int N = 30; // puedes cambiar para refinamiento
mesh Th = square(N, N);

// 2. Espacio de elementos finitos P1
fespace Vh(Th, P1);
Vh u, v;

// 3. Función fuente
func f = 2*pi^2*sin(pi*x)*sin(pi*y);

// 4. Problema variacional
problem Poisson(u, v) =
    int2d(Th)( dx(u)*dx(v) + dy(u)*dy(v) )
  - int2d(Th)( f * v )
  + on(1,2,3,4, u=0);

// 5. Resolver
Poisson;

// 6. Visualizar solución
plot(u, fill=true, value=true, wait=1);

// ─── Cálculo de errores ───────────────────────────────
// Solución exacta
func uex = sin(pi*x)*sin(pi*y);

// Derivadas exactas (forma eficiente)
func uexdx = pi*cos(pi*x)*sin(pi*y);
func uexdy = pi*sin(pi*x)*cos(pi*y);

// Error L2
real errL2 = sqrt(int2d(Th)( (u - uex)^2 ));

// Error H1 (solo gradiente)
real errH1 = sqrt(
    int2d(Th)(
        (dx(u) - uexdx)^2 +
        (dy(u) - uexdy)^2
    )
);

// Error H1 completo (opcional, más riguroso)
real errH1full = sqrt(
    int2d(Th)(
        (u - uex)^2 +
        (dx(u) - uexdx)^2 +
        (dy(u) - uexdy)^2
    )
);

// 7. Resultados
cout << "Malla: " << N << "x" << N << endl;
cout << "||u-uh||_L2 = " << errL2 << endl;
cout << "||u-uh||_H1 (gradiente) = " << errH1 << endl;
cout << "||u-uh||_H1 (completo)  = " << errH1full << endl;

Refinamiento Adaptativo de Malla

El refinamiento uniforme es caro: refinar todo para mejorar el error en una esquina es un desperdicio. El refinamiento adaptativo concentra los elementos donde el error es grande.

El problema del refinamiento uniforme

Para P1 uniforme, reducir el error a la mitad requiere duplicar los nodos en cada dirección:

Costo computacional $$\|u - u_h\|_{H^1} \leq C\,h \,|u|_{H^2} \quad \Rightarrow \quad h \to h/2 \Rightarrow N \to 4N \text{ (2D)}$$

Si la solución tiene singularidades (esquinas reentrantes, cambios de coeficiente), el error se concentra localmente. Refinar globalmente malgasta el 90% del esfuerzo.

Ventaja del refinamiento adaptativo
Tasa óptima $$\|u - u_h\|_{H^1} \leq C\, N_h^{-1/2} \quad \text{(adaptativo, 2D)}$$ $$\|u - u_h\|_{H^1} \leq C\, N_h^{-1/(2+\varepsilon)} \quad \text{(uniforme, singularidad)}$$

El refinamiento adaptativo recupera la tasa óptima incluso en presencia de singularidades, concentrando nodos solo donde el estimador de error es grande.

Estimador a posteriori de Zienkiewicz-Zhu (ZZ)

Una vez resuelta $u_h$, estimamos el error localmente en cada elemento $K$:

Estimador de salto (residual) $$\eta_K^2 = h_K^2 \|f + \Delta u_h\|_{L^2(K)}^2 + h_K \sum_{E \subset \partial K} \left\| \left[\frac{\partial u_h}{\partial n}\right] \right\|_{L^2(E)}^2$$

$\left[\frac{\partial u_h}{\partial n}\right]$ es el salto de la derivada normal de $u_h$ en la arista $E$ entre dos elementos. Para P1, $\Delta u_h = 0$ en cada $K$, así que solo cuenta el salto.

Confiabilidad y eficiencia: El estimador $\eta = \left(\sum_K \eta_K^2\right)^{1/2}$ satisface: $$C_1\,\eta \leq \|u - u_h\|_{H^1(\Omega)} \leq C_2\,\eta$$ Es decir, controla el error real tanto por arriba como por abajo (salvo términos de oscilación de datos).

El estimador ZZ más simple promedia el gradiente en nodos: $\eta_K = \|G_h(\nabla u_h) - \nabla u_h\|_{L^2(K)}$ donde $G_h$ es un operador de recuperación suave. Es barato y funciona muy bien en la práctica.

Criterio de marcado (Dörfler / bulk criterion) $$\text{Marcar conjunto minimal } \mathcal{M} \subset \mathcal{T}_h \text{ tal que:} \quad \sum_{K \in \mathcal{M}} \eta_K^2 \geq \theta^2 \sum_{K \in \mathcal{T}_h} \eta_K^2, \quad \theta \in (0,1)$$

Un valor típico es $\theta = 0.5$. Valores grandes ($\theta \to 1$) refuerzan más agresivamente; valores pequeños ($\theta \to 0$) refinan más conservadoramente.

Ciclo adaptativo en FreeFEM++
FreeFEM++ / adaptativo
// ─── Refinamiento Adaptativo · FreeFEM++ ─────────────────
// Ciclo SOLVE → ESTIMATE → MARK → REFINE
mesh Th = square(5, 5);   // malla inicial gruesa
func f  = 2*pi^2*sin(pi*x)*sin(pi*y);
func uex = sin(pi*x)*sin(pi*y);

for(int iter = 0; iter < 8; iter++){

  // ── 1. SOLVE ─────────────────────────────────────────
  fespace Vh(Th, P1);
  Vh u, v;
  problem Poisson(u, v) =
      int2d(Th)( dx(u)*dx(v) + dy(u)*dy(v) )
    - int2d(Th)( f * v )
    + on(1,2,3,4, u=0);
  Poisson;

  // ── 2. ESTIMATE (error L2 local por elemento) ────────
  fespace Wh(Th, P0);   // espacio de constantes por elementos
  Wh err;
  err = int2d(Th)((u - uex)^2) / area;

  // ── 3. MARK + REFINE (adaptmesh usa el estimador) ────
  Th = adaptmesh(Th, u,
         err   = 1e-3,    // tolerancia de error
         nbvx  = 50000,   // máx. vértices
         ratio = 1.8,     // ratio de gradación
         iso   = 0);      // refinamiento anisotrópico
  plot(Th);

  // ── 4. Reportar progreso ─────────────────────────────
  real errL2 = sqrt(int2d(Th)((u - uex)^2));
  cout << "Iter " << iter << " | nv=" << Th.nv
       << " | errL2=" << errL2 << endl;
}
